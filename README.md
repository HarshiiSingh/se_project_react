# WTWR (What to Wear?)

This is the newest project of the Software Engineering program at TripleTen. It is created using React, Vite.js, and CSS, based on a Figma design brief.

## Project Features

- React
- Flexbox
- Positioning
- Dynamic Grid Layout
- Vite.js
